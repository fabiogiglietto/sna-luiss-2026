Setup Strumenti

Completa questi passaggi prima della lezione. Tutti gli strumenti sono gratuiti.

Installa R

Vai su cran.r-project.org e scarica la versione per il tuo sistema operativo. Versione consigliata: R 4.3+

Installa RStudio

Vai su posit.co e scarica RStudio Desktop (gratuito).

Alternativa: Usa posit.cloud per RStudio nel browser (nessuna installazione, piano gratuito 25h/mese).

Installa i pacchetti R

Apri RStudio e incolla questo codice nella Console:

# Pacchetti per il corso SNA
install.packages(c(
  "igraph",     # analisi reti
  "ggraph",     # visualizzazione
  "tidyverse",  # manipolazione dati
  "readr",      # import CSV
  "corrplot",   # matrici correlazione
  "ggpubr"      # grafici
))

Installa Gephi

Vai su gephi.org e scarica la versione per il tuo OS. Richiede Java 11+.

Verifica l'installazione

Scarica ed esegui setup_verifica.R in RStudio. Se vedi "Setup completato con successo!", sei pronto!

Problemi? Contatta il docente prima della lezione con uno screenshot dell'errore.
